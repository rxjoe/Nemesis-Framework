# Nemesis Framework v3.0 Ultimate

أقوى سكربت كشف ثغرات لـ Roblox — 200+ تقنية كشف

## 📥 التشغيل

انسخ هذا الكود في الـ executor بتاعك:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/rxjoe/Nemesis-Framework/main/loader.lua"))()
```

## 🎯 الاستخدام

1. ادخل أي ماب Roblox
2. حط الكود فوق في الـ executor
3. اضغط على **"START FULL SCAN"**
4. استنى الفحص يخلص
5. اضغط **"COPY ALL"** عشان تنسخ كل الثغرات

## 📋 المميزات

- 200+ تقنية كشف ثغرات
- 19 قسم تصنيف
- واجهة احترافية بالتابات
- Copy All بنقرة واحدة
- بدون hitbox أو فلوس — ثغرات فقط
